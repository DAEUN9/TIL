- 중복을 허용하면서 저장순서가 유지되는 컬렉션을 구현하는데 사용

- 중복 X, 저장순서가 유지 X



### Map인터페이스

- 키 값을 하나의 쌍으로 묶어서 저장하는 컬렉션 클래스 구현
